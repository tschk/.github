# The Software Company of Hong Kong

An independent R&D lab building systems software, runtimes, developer tools, application platforms, and experimental computing platforms.

## Projects

- [crepuscularity](https://crepuscularity.tsc.hk) — hardware-accelerated UI for native apps, web, terminals, and embedded appliances
- [alpenglow](https://alpenglow.tsc.hk) — a Linux operating system that boots in under a second
- [inauguration](https://inauguration.tsc.hk) — deterministic execution and agent-native software
- [rotary](https://github.com/tschk/rotary) — a pure Rust agent harness engine for hosts, tools, permissions, sessions, and control planes
- [nexnet](https://nexnet.tsc.hk) — a local-first, peer-to-peer social chat network with wallet identity and encrypted messaging
